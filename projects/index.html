PROJECTS

A collection of some of my projects. Click on the links below to access individual projects. Here is the link to my GitHub: https://github.com/Ilyankor. Please view the GitHub page for more.

Phoneme learning

Developed a basic nueral network to learn English phonemic structure from public domain texts. The generated text was used in a song.

Windswept Banner

An electronic piece for fixed media using data from numerically solved partial differential equation. Earthquake seismogram data was interpreted as audio. Animation of solution of the partial differential equation with FFMPEG.

View the animation on YouTube

Marbles in a bag

Using discrete probability distributions, random choices were made and determined various aspects of music like pitch and rhythm. Additionally, Markov chains were used to determine successive pitches. This was the first time I used probability and computer assistsance to generate music. For more information about the process, visit the GitHub repository.

Heart animation

Animation of an SVG using CSS. An early project in HTML/CSS.
